Săptămâna 4 – JavaScript de bază

1. Introducere în JavaScript

JavaScript este un limbaj de programare care permite adăugarea de interactivitate paginilor web. Poți folosi JavaScript pentru a manipula conținutul HTML, a răspunde la evenimente ale utilizatorului și a crea aplicații dinamice. În această săptămână, vom acoperi conceptele de bază: variabile, funcții și manipularea DOM-ului.

2. Adăugarea JavaScript în pagini HTML

Pentru a folosi JavaScript într-o pagină HTML, poți adăuga codul în mai multe moduri. Cele mai comune sunt: inline, în elementul <script>, sau într-un fișier extern.

Metode de adăugare

  • Inline: Codul JS este scris direct în atributul unui element HTML (de exemplu, onclick).
  • În elementul <script>: Codul este plasat între tag-urile <script> și </script>, de obicei la sfârșitul <body>.
  • Fișier extern: Codul este într-un fișier separat (.js) și este inclus folosind atributul src.

Exemple

<!DOCTYPE html>
<html lang="ro">
<head>
    <meta charset="UTF-8">
    <title>Exemplu JS</title>
</head>
<body>
    <button onclick="alert('Salut!')">Click inline</button>

    <script>
        // Cod JS în script tag
        console.log("Acesta este JS în pagină");
    </script>

    <script src="script.js"></script> <!-- Fișier extern -->
</body>
</html>

Recomandarea este să plasezi script-urile la sfârșitul <body> pentru a nu bloca încărcarea paginii.

3. Variabile

Variabilele sunt containere pentru stocarea datelor. În JavaScript, poți declara variabile folosind var, let sau const.

Tipuri de variabile

  • var – Declarație tradițională, dar nu este recomandată în codul modern din cauza problemelor de scope.
  • let – Permite reatribuirea valorii și are scope de bloc.
  • const – Pentru valori constante care nu se schimbă; are scope de bloc și nu permite reatribuirea.

Exemple

// Declarare cu let
let nume = "Ana";
nume = "Maria"; // OK, se poate schimba

// Declarare cu const
const varsta = 25;
// varsta = 26; // Eroare: nu se poate schimba

// Declarare cu var (nu recomandat)
var oras = "București";
oras = "Cluj"; // OK, dar scope global

4. Funcții

Funcțiile sunt blocuri de cod reutilizabile care execută sarcini specifice. Pot primi parametri și pot returna valori.

Tipuri de funcții

  • Funcții declarate: Sunt definite folosind cuvântul cheie function.
  • Funcții expresie: Sunt atribuite unei variabile.
  • Funcții săgeată (arrow functions): Sintaxă scurtă, introdusă în ES6.

Exemple

// Funcție declarată
function salut(nume) {
    return "Salut, " + nume + "!";
}

// Funcție expresie
const aduna = function(a, b) {
    return a + b;
};

// Funcție săgeată
const inmulteste = (a, b) => a * b;

// Apelarea funcțiilor
console.log(salut("Ana")); // Salut, Ana!
console.log(aduna(2, 3)); // 5
console.log(inmulteste(4, 5)); // 20

5. Manipularea DOM-ului

DOM (Document Object Model) reprezintă structura paginii HTML ca un arbore de obiecte. JavaScript poate manipula DOM-ul pentru a schimba conținutul, stilurile sau structura paginii.

Selectarea elementelor

Pentru a manipula elemente, trebuie să le selectezi mai întâi.

  • document.getElementById(id) – Selectează un element după ID.
  • document.querySelector(selector) – Selectează primul element care corespunde selectorului CSS.
  • document.querySelectorAll(selector) – Selectează toate elementele care corespund selectorului.

Exemple de manipulare

// Selectarea unui element
const titlu = document.querySelector('h1');

// Schimbarea textului
titlu.textContent = "Titlu nou";

// Schimbarea stilului
titlu.style.color = "red";

// Adăugarea unui eveniment
titlu.addEventListener('click', function() {
    alert('Ai făcut click pe titlu!');
});

// Crearea și adăugarea unui element nou
const paragrafNou = document.createElement('p');
paragrafNou.textContent = "Acesta este un paragraf nou.";
document.body.appendChild(paragrafNou);

Exemplu practic: Buton care schimbă culoarea fundalului

Poți combina variabile, funcții și DOM pentru a crea interacțiuni simple.

<button id="schimbaCuloare">Schimbă culoarea</button>

<script>
const buton = document.getElementById('schimbaCuloare');
let culoareCurenta = 'white';

buton.addEventListener('click', function() {
    culoareCurenta = (culoareCurenta === 'white') ? 'lightblue' : 'white';
    document.body.style.backgroundColor = culoareCurenta;
});
</script>

6. Evenimente și interactivitate

Evenimentele permit paginilor să răspundă la acțiunile utilizatorilor, cum ar fi click-uri, apăsări de taste sau mișcări ale mouse-ului. JavaScript folosește ascultători de evenimente (event listeners) pentru a detecta și reacționa la aceste evenimente.

Evenimente comune

  • click – Când utilizatorul face click pe un element.
  • mouseover – Când mouse-ul trece peste un element.
  • keydown – Când o tastă este apăsată.
  • submit – Când un formular este trimis.
  • load – Când pagina s-a încărcat complet.

Exemple de evenimente

// Eveniment click
const buton = document.querySelector('button');
buton.addEventListener('click', function() {
    alert('Butonul a fost apăsat!');
});

// Eveniment mouseover
const div = document.querySelector('div');
div.addEventListener('mouseover', function() {
    div.style.backgroundColor = 'yellow';
});

// Eveniment keydown
document.addEventListener('keydown', function(event) {
    console.log('Tasta apăsată: ' + event.key);
});

// Eveniment submit pentru formular
const form = document.querySelector('form');
form.addEventListener('submit', function(event) {
    event.preventDefault(); // Previne trimiterea implicită
    alert('Formular trimis!');
});

Exemplu practic: Formular interactiv

Un formular simplu care validează input-ul și afișează un mesaj.

<form id="myForm">
    <input type="text" id="nume" placeholder="Introdu numele tău">
    <button type="submit">Trimite</button>
</form>
<p id="mesaj"></p>

<script>
const form = document.getElementById('myForm');
const input = document.getElementById('nume');
const mesaj = document.getElementById('mesaj');

form.addEventListener('submit', function(event) {
    event.preventDefault();
    if (input.value.trim() === '') {
        mesaj.textContent = 'Te rog introdu un nume!';
        mesaj.style.color = 'red';
    } else {
        mesaj.textContent = 'Salut, ' + input.value + '!';
        mesaj.style.color = 'green';
    }
});
</script>
