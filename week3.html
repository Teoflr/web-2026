Săptămâna 3 – Tabele HTML, Media Queries și Design Responsiv

1. Introducere în tabele HTML

Tabelele HTML sunt folosite pentru a organiza date într-un format structurat, cu rânduri și coloane. Ele sunt utile pentru afișarea informațiilor tabulare, cum ar fi liste de produse, orare sau date statistice. Elementele de bază ale unui tabel includ <table> pentru containerul principal, <tr> pentru rânduri, <td> pentru celulele de date și <th> pentru antetele coloanelor.

Structura de bază a unui tabel

Un tabel simplu constă din rânduri (<tr>) care conțin celule. Prima rând poate fi dedicată antetelor (<th>), iar celelalte rânduri conțin date (<td>).

Nume Vârstă Oraș
Ana 25 București
Mihai 30 Cluj 
<table>
    <tr>
        <th>Nume</th>
        <th>Vârstă</th>
        <th>Oraș</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>25</td>
        <td>București</td>
    </tr>
    <tr>
        <td>Mihai</td>
        <td>30</td>
        <td>Cluj</td>
    </tr>
</table>

Atribute utile pentru tabele

  • colspan – Permite unei celule să se întindă pe mai multe coloane. De exemplu, <td colspan="2"> îmbină două coloane.
  • rowspan – Permite unei celule să se întindă pe mai multe rânduri. De exemplu, <td rowspan="2"> îmbină două rânduri.
  • border – Adaugă o bordură tabelului (deși este mai bine să folosești CSS pentru stilizare).

Exemplu cu colspan și rowspan

Ziua Activitate Durată
Luni Curs HTML 2 ore
Curs CSS 1 oră
Marți Proiect practic 
<table>
    <tr>
        <th>Ziua</th>
        <th>Activitate</th>
        <th>Durată</th>
    </tr>
    <tr>
        <td rowspan="2">Luni</td>
        <td>Curs HTML</td>
        <td>2 ore</td>
    </tr>
    <tr>
        <td>Curs CSS</td>
        <td>1 oră</td>
    </tr>
    <tr>
        <td>Marți</td>
        <td colspan="2">Proiect practic</td>
    </tr>
</table>

Stilizarea tabelelor cu CSS

Pentru a face tabelele mai atractive, poți adăuga stiluri CSS. De exemplu, poți seta borduri, culori de fundal pentru antete și spațiere pentru celule.

table {
    border-collapse: collapse;
    width: 100%;
}
th, td {
    border: 1px solid #ddd;
    padding: 8px;
    text-align: left;
}
th {
    background-color: #f2f2f2;
}

2. Introducere în Media Queries

Media queries sunt o caracteristică CSS care permite aplicarea stilurilor diferite în funcție de caracteristicile dispozitivului, cum ar fi lățimea ecranului, înălțimea, orientarea (portret sau peisaj) sau tipul de dispozitiv (ecran, imprimantă). Ele sunt esențiale pentru designul responsiv, permițând site-urilor să se adapteze la diferite mărimi de ecran.

Sintaxa de bază

O media query începe cu @media, urmată de o condiție între paranteze și un bloc de stiluri între acolade.

@media (max-width: 768px) {
    body {
        font-size: 14px;
    }
}

Condiții comune în media queries

  • max-width – Aplică stilurile dacă lățimea ecranului este mai mică sau egală cu valoarea specificată.
  • min-width – Aplică stilurile dacă lățimea ecranului este mai mare sau egală cu valoarea specificată.
  • orientation: portrait – Pentru dispozitive în modul portret (înălțime mai mare decât lățimea).
  • orientation: landscape – Pentru dispozitive în modul peisaj (lățime mai mare decât înălțimea).

Exemplu de media query pentru dispozitive mobile

@media (max-width: 600px) {
    .container {
        flex-direction: column;
    }
    nav ul {
        flex-direction: column;
    }
}

Breakpoint-uri populare

Breakpoint-urile sunt punctele în care layout-ul se schimbă. Unele valori comune sunt:

  • 320px – Telefoane mici
  • 768px – Tablete
  • 1024px – Laptop-uri mici
  • 1200px – Desktop-uri

3. Introducere în crearea unei pagini HTML responsive

Designul responsiv înseamnă crearea paginilor web care se adaptează automat la diferite mărimi de ecran, oferind o experiență bună pe toate dispozitivele. Principiul de bază este "mobile-first": proiectează mai întâi pentru dispozitivele mobile, apoi adaugă stiluri pentru ecrane mai mari.

Principiile designului responsiv

  • Layout flexibil – Folosește unități relative (%, em, rem, vh, vw) și proprietăți CSS care permit elementelor să se redimensioneze automat. De exemplu, cu display: flex; transformi un container într-unul în care copiii ocupă spațiul disponibil. Proprietăți utile pentru layout flexibil:
    • flex-wrap: wrap; – permite elementelor să treacă pe rând nou când nu mai încap pe o linie.
    • flex: 1 1 200px; – combină flex-grow, flex-shrink și flex-basis, astfel încât elementele să crească, să se micșoreze și să pornească de la o dimensiune de bază.
    • max-width: 100%; – util pentru imagini, tabele și alte elemente pentru a nu depăși containerul părinte.
    • box-sizing: border-box; – face ca padding-ul și border-ul să fie incluse în dimensiunile elementului, astfel încât calculele de lățime/înălțime să fie mai previzibile.
  • Imagini flexibile – Setează max-width: 100% pentru imagini pentru a le împiedica să depășească containerul.
  • Media queries – Aplică stiluri diferite pentru diferite breakpoint-uri.
  • Grid și Flexbox – Folosește aceste tehnici pentru layout-uri care se adaptează.

Un exemplu practic de layout flexibil este folosirea unei clase container cu display: flex; și setarea flex pentru elementele copil. Astfel, elementele se redimensionează proporțional și se pot rearanja în funcție de spațiu.

Detalii despre proprietatea flex și subproprietăți

flex este un shorthand care combină trei proprietăți:

  • flex-grow – cât de mult crește un element pentru a umple spațiul liber (valoarea default este 0).
  • flex-shrink – cât de mult se micșorează un element atunci când nu mai încap toate elementele într-un rând (valoarea default este 1).
  • flex-basis – dimensiunea de bază a elementului înainte de a împărți spațiul (poate fi auto, 0, sau o valoare fixă precum 200px).

Exemplu: flex: 1 1 220px; înseamnă că elementul pornește de la 220px, poate crește (dacă mai este spațiu) și se poate micșora (dacă spațiul scade).

.container {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}
.card {
    flex: 1 1 220px; /* crește, se micșorează, pornește de la 220px */
    min-width: 200px;
}
img {
    max-width: 100%;
    height: auto;
}

Exemplu de pagină responsive simplă

Următorul exemplu arată un layout cu două coloane care devine o singură coloană pe dispozitive mici.

<!DOCTYPE html>
<html lang="ro">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pagină Responsive</title>
    <style>
        .container {
            display: flex;
            gap: 1rem;
        }
        .sidebar {
            flex: 1;
        }
        .main {
            flex: 3;
        }
        @media (max-width: 768px) {
            .container {
                flex-direction: column;
            }
        }
    </style>
</head>
<body>
    <div class="container">
        <aside class="sidebar">Bară laterală</aside>
        <main class="main">Conținut principal</main>
    </div>
</body>
</html>

Meta viewport

Pentru ca pagina să fie responsive, trebuie să adaugi <meta name="viewport" content="width=device-width, initial-scale=1.0"> în head. Acest lucru spune browserului să seteze lățimea viewport-ului la lățimea dispozitivului și să prevină zoom-ul inițial.

Exercițiu practic

Construiește o pagină HTML care include:

  1. Un tabel cu cel puțin 5 coloane (de exemplu: curs, profesor, durată, nivel, status).
  2. O clasă CSS pentru tabel (.responsive-table) care adaugă borduri, spațiere și culori ușoare pentru antete.
  3. Cel puțin un media query care modifică layout-ul pe mobil:
    • Pe ecrane mici (max-width: 600px) ascunde coloanele mai puțin importante (de exemplu durata și status).
    • Sau transformă rândurile în „carduri” (folosind display: block și display: grid pentru fiecare rând) pentru o vizualizare mai ușor de citit.

Test: Deschide pagina în browser și redimensionează fereastra de la desktop la mobil; verifică că tabelul devine mai ușor de citit și că coloanele se ascund / se rearanjează corect.
