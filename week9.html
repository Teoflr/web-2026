Laborator 9 – MySQL si phpMyAdmin: crearea unei baze de date si aplicatie CRUD pentru studenti

1. Pornirea XAMPP si accesarea phpMyAdmin

  1. Porniti Apache si MySQL din XAMPP Control Panel.
  2. Deschideti browser-ul si mergeti la http://localhost/phpmyadmin.
  3. Interfata phpMyAdmin permite administrarea vizuala a bazelor de date.

2. Crearea bazei de date si a tabelei "studenti"

O baza de date contine tabele. Un tabel are coloane (câmpuri) si randuri (inregistrari).

Executati urmatorul script SQL in fila "SQL" din phpMyAdmin:

CREATE DATABASE facultate;
USE facultate;

CREATE TABLE studenti (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nume VARCHAR(100) NOT NULL,
    prenume VARCHAR(100) NOT NULL,
    grupa VARCHAR(20) NOT NULL,
    an_studiu INT NOT NULL,
    specializare VARCHAR(100)
);

Explicatii coloane: id cheie primara, creste automat. NOT NULL asigura completarea campului. VARCHAR(100) = sir de maxim 100 caractere.

3. Ce este un query SQL?

Query-urile sunt comenzile trimise bazei de date pentru manipulare date.

  • INSERT: adauga un rand nou. Exemplu: INSERT INTO studenti (nume, prenume, grupa, an_studiu) VALUES ('Popescu', 'Ion', 'A1', 2);
  • SELECT: extrage date. SELECT * FROM studenti WHERE an_studiu = 1;
  • UPDATE: modifica date existente. UPDATE studenti SET grupa = 'A2' WHERE id = 3;
  • DELETE: sterge randuri. DELETE FROM studenti WHERE id = 5;

In PHP vom folosi functii mysqli pentru a executa aceste comenzi in siguranta.

4. Conectarea PHP la MySQL (mysqli)

Cream fisierul config.php in folderul C:\xampp\htdocs\student_crud (sau similar pe macOS).

<?php
// config.php
$host = 'localhost';
$user = 'root';
$password = '';
$database = 'facultate';

$conn = new mysqli($host, $user, $password, $database);

if ($conn->connect_error) {
    die('Conexiune esuata: ' . $conn->connect_error);
}
$conn->set_charset('utf8');
?>

5. Formular de adaugare studenti si afisare tabel

Cream fisierul index.php care contine formularul (metoda POST) si tabelul cu toti studentii din baza de date.

<?php
require_once 'config.php';

// Procesare adaugare student
if ($_SERVER['REQUEST_METHOD'] === 'POST' && isset($_POST['adauga'])) {
    $nume = trim($_POST['nume']);
    $prenume = trim($_POST['prenume']);
    $grupa = trim($_POST['grupa']);
    $an_studiu = (int)$_POST['an_studiu'];
    $specializare = trim($_POST['specializare']);

    if (!empty($nume) && !empty($prenume) && !empty($grupa) && $an_studiu >= 1 && $an_studiu <= 4) {
        $stmt = $conn->prepare("INSERT INTO studenti (nume, prenume, grupa, an_studiu, specializare) VALUES (?, ?, ?, ?, ?)");
        $stmt->bind_param("sssis", $nume, $prenume, $grupa, $an_studiu, $specializare);
        if ($stmt->execute()) {
            $succes = "Student adaugat cu succes.";
        } else {
            $eroare = "Eroare la inserare: " . $conn->error;
        }
        $stmt->close();
    } else {
        $eroare = "Toate campurile obligatorii trebuie completate corect.";
    }
}

// Preluare studenti pentru afisare
$sql = "SELECT * FROM studenti ORDER BY id DESC";
$result = $conn->query($sql);
?>
<!DOCTYPE html>
<html lang="ro">
<head>
    <meta charset="UTF-8">
    <title>Gestionare studenti</title>
    <style> table, th, td { border: 1px solid #ccc; border-collapse: collapse; padding: 6px; } </style>
</head>
<body>
    <h2>Adauga student nou</h2>
    <?php if(isset($succes)) echo "<p style='color:green'>$succes</p>"; ?>
    <?php if(isset($eroare)) echo "<p style='color:red'>$eroare</p>"; ?>
    <form method="post">
        Nume: <input type="text" name="nume" required><br>
        Prenume: <input type="text" name="prenume" required><br>
        Grupa: <input type="text" name="grupa" required><br>
        An studiu: <input type="number" name="an_studiu" min="1" max="4" required><br>
        Specializare: <input type="text" name="specializare"><br>
        <button type="submit" name="adauga">Adauga student</button>
    </form>

    <h2>Lista studenti</h2>
    <?php if ($result && $result->num_rows > 0): ?>
    <table>
        IDNumePrenumeGrupaAnSpecializare
        <?php while($row = $result->fetch_assoc()): ?>
        
            <?= $row['id'] ?>
            <?= htmlspecialchars($row['nume']) ?>
            <?= htmlspecialchars($row['prenume']) ?>
            <?= htmlspecialchars($row['grupa']) ?>
            <?= $row['an_studiu'] ?>
            <?= htmlspecialchars($row['specializare']) ?>
        
        <?php endwhile; ?>
    
    <?php else: ?>
        <p>Nu exista studenti in baza de date.</p>
    <?php endif; ?>
    <?php $conn->close(); ?>
</body>
</html>

6. Testare si depanare

Accesati http://localhost/student_crud/index.php. Completati formularul. Studentii nou adaugati vor aparea imediat in tabel.

Daca nu se conecteaza, verificati daca MySQL ruleaza. Verificati numele bazei de date si credentialele.
