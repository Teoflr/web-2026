Săptămâna 2 – HTML & CSS (partea 1)

Ce înseamnă structura unei pagini

O pagină web este alcătuită din tag-uri HTML care definesc semnificaţia fiecărei secţiuni. Iată o listă de elemente de bază:

  • <!DOCTYPE html> – definește versiunea HTML; spune browserului că documentul este HTML5.
  • <html> – elementul root ce înconjoară tot codul; include atributul lang pentru limba paginii.
  • <head> – secțiune pentru metadate (de exemplu <meta charset="UTF-8">), titlu, linkuri către fișiere CSS sau scripturi JavaScript; nu apare direct în conținutul vizibil.
  • <body> – partea vizibilă a paginii; conține text, imagini, formulare și alte elemente pe care utilizatorul le vede și interacționează cu ele.

Exemplu de pagină cu două coloane și meniu

<!DOCTYPE html>
<html lang="ro">
<head>
    <meta charset="UTF-8">
    <title>Site simplu</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Titlul site‑ului</h1>
        <nav>
            <ul>
                <li><a href="#">Acasă</a></li>
                <li><a href="#">Despre</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
    </header>
    <div class="container">
        <aside>Aceasta este coloana laterală.</aside>
        <main>Aici stă conţinutul principal.</main>
    </div>
    <footer>© 2026 Exemplu</footer>
</body>
</html>

Explicaţii pentru tag-urile din exemplu

  • <header> – zona superioară a paginii, de obicei conține titluri și meniuri.
  • <h1> – titlul principal al paginii; Este permis doar un singur h1 pe pagină.
  • <nav> – element semantic pentru blocul de navigare.
  • <ul> și <li> – listă neordonată și element de listă, folosite pentru meniuri.
  • <a href="..."> – link; atributul href indică adresa destinatară.
  • <div class="container"> – divizare generică utilizată aici pentru a grupa coloană laterală și conținutul principal.
  • <aside> – secţiune secundară, de obicei pentru note, legături sau informaţii adiţionale.
  • <main> – conţinutul principal al paginii; nu trebuie plasat în interiorul altor elemente precum header sau footer.
  • <footer> – partea de jos a paginii, folosită pentru informaţii de subsol sau copyright.

Baze CSS

CSS (Cascading Style Sheets) se aplică elementelor HTML folosind selectori care identifică taguri, clase sau id-uri. O regulă CSS are trei părţi principale:

  1. Selector – elementul căruia i se aplică stilul (ex. body, nav a, .container).
  2. Proprietate – ce aspect vrei să modifici (ex. color, margin, display).
  3. Valoare – ce setare aplici proprietăţii (ex. #333, 1rem, flex).

Exemplu de sintaxă:

selector {
    proprietate: valoare;
    /* comentariu: această linie este ignorată */
}

Comentariile în CSS sunt scrise între /* și */ şi nu influenţează stilurile.

CSS pentru Începători – Ghid Esențial

1. Ce este CSS și cum funcționează

  • CSS (Cascading Style Sheets) este limbajul folosit pentru stilizarea paginilor HTML.
  • Separă conținutul (HTML) de design (culori, layout, fonturi).
  • Funcționează pe principiul „cascadă”: unele reguli le pot suprascrie pe altele în funcție de prioritate (specificitate și ordine).

2. Cum se adaugă CSS într-o pagină

Inline

<p style="color: red;">Text</p>

Internal

<head>
  <style>
    p { color: red; }
  </style>
</head>

External (recomandat)

<link rel="stylesheet" href="style.css">

3. Sintaxa de bază

selector {
  proprietate: valoare;
}

Exemplu:

p {
  color: blue;
  font-size: 16px;
}

4. Selectori

  • Selector de element: p, div, h1
  • Selector de clasă: .container
  • Selector de ID: #header
  • Selectori descendenți: div p
  • Selectori copil direct: ul > li
  • Selectori multipli: h1, h2, h3

5. Culori și unități de măsură

Culori

  • Nume: red
  • Hex: #ff0000
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5)

Unități

  • px – unitate absolută, pixeli. Dimensiunea nu se schimbă indiferent de context. Folosit pentru comportament fix și predictibil.
  • % – procent relativ la elementul părinte. Dacă părintele are 200px lățime și copilul are 50%, copilul va fi 100px. Util pentru layout responsive.
  • em – relativ la dimensiunea fontului elementului curent. Dacă fontul este 16px și setezi margin: 1em, va fi 16px. Dacă schimbi font-size la 20px, 1em devine 20px.
  • rem – relativ la dimensiunea fontului elementului root (<html>). Spre deosebire de em, nu depinde de font-size părintelui. Recomandată pentru layout consistent.
  • vh – viewport height, procent din înălțimea ecranului. 100vh = înălțimea întregului ecran. Util pentru secțiuni pe întreg ecranul.
  • vw – viewport width, procent din lățimea ecranului. 100vw = lățimea întregului ecran. Util pentru layout responsive pe mai multe dispozitive.

6. Modelul Box (Box Model)

Fiecare element HTML este o „cutie” formată din:

  • content – zona centrală unde se găsește textul, imaginile și alte conținut. Dimensiunea se controlează cu proprietățile width și height.
  • padding – spațiul interior dintre conținut și margine. Crește spațiul din interiorul elementului, dar nu afectează elementele din jur. Exemplu: padding: 1rem;.
  • border – linia de contur care înconjoară padding-ul. Se controlează cu proprietăți precum border: 2px solid #333;. Are grosime, stil și culoare.
  • margin – spațiul exterior dintre element și elementele din jur. Crește distanța dintre acest element și alții. Exemplu: margin: 1rem;.

Ordinea de la centru spre exterior: content → padding → border → margin

Recomandare:

* {
  box-sizing: border-box;
}

Aceasta face ca width și height să includă padding-ul și border-ul, ceea ce te ajută să calculezi cu ușurință dimensiunile elementelor.

7. Display și poziționare

Display

  • block – elementul ocupă toată lățimea disponibilă și forțează un rând nou. Exemplu: <div>, <p>, <h1>. Margin-ul și padding-ul funcționează pe toate laturile.
  • inline – elementul ocupă doar spațiul necesar și nu forțează rând nou. Alte elemente se pot alinia pe aceeași linie. Exemplu: <span>, <a>. Margin-ul vertical nu funcționează.
  • inline-block – combinație dintre inline și block. Elementul se aliniază inline cu alții, dar proprietățile de margin și padding funcționează pe toate laturile. Util pentru alinierea mai multor cutii pe o linie.
  • none – elementul este complet ascuns și nu ocupă spațiu în pagină. Folosit pentru ascunderea elementelor fără a le șterge din HTML.

Poziționare

  • static – valoarea implicită. Elementul urmează fluxul normal al documentului. Proprietățile top, bottom, left, right nu au efect.
  • relative – elementul se poziționează relativ la poziția sa normală. Poți folosi top, bottom, left, right pentru a-l muta. Ocupă încă spațiu în fluxul normal.
  • absolute – elementul se scoate din fluxul normal și se poziționează relativ la cel mai apropiat element părinte care are position setat (nu static). Dacă nu există, se poziționează relativ la ferestrele browserului. Ocupă zero spațiu în flux.
  • fixed – elementul se poziționează relativ la fereastra browserului și rămâne pe același loc chiar dacă faci scroll. Folosit pentru bare de navigare sau butoane floating. Se scoate din fluxul normal.
  • sticky – element hibrid care se comportă ca static până când ajunge la o anumită poziție (definită cu top, bottom), după care se comportă ca fixed. Util pentru titluri care urmăresc scrollul.

8. Flexbox (layout modern)

Activare:

.container {
  display: flex;
}

Proprietăți principale:

  • flex-direction – stabilește direcția elementelor (row, column).
  • justify-content – aliniază elementele pe axa principală (orizontal implicit).
  • align-items – aliniază elementele pe axa secundară (vertical implicit).
  • align-content – aliniază liniile când există mai multe rânduri.
  • flex-wrap – permite elementelor să treacă pe rând nou.
  • gap – spațiu între elemente.
  • flex-grow – permite unui element să ocupe spațiu suplimentar.
  • flex-shrink – permite micșorarea unui element dacă e nevoie.
  • flex-basis – dimensiunea inițială a unui element.

9. Grid (layout complex)

Activare:

.container {
  display: grid;
}

Proprietăți principale:

  • grid-template-columns – definește numărul și dimensiunea coloanelor.
  • grid-template-rows – definește numărul și dimensiunea rândurilor.
  • gap – spațiu între rânduri și coloane.
  • grid-column – stabilește pe ce coloane se întinde un element.
  • grid-row – stabilește pe ce rânduri se întinde un element.
  • grid-area – definește poziția unui element într-o zonă.
  • justify-items – aliniază elementele pe orizontală în celule.
  • align-items – aliniază elementele pe verticală în celule.
  • place-items – combină justify-items și align-items.

14. Practici bune

  • Folosește clase în loc de ID pentru stilizare.
  • Evită !important.
  • Structurează codul clar și logic.
  • Grupează proprietățile similare.
  • Comentează codul.
  • Folosește denumiri clare pentru clase (ex: card, nav-item).
  • Menține consistența în indentare.
  • Testează în mai multe browsere.
  • Folosește box-sizing: border-box; global.
/* reset minimal */
body, h1, h2, h3, p, ul, li {
    margin: 0;
    padding: 0;
}
body {
    font-family: Arial, sans-serif;
}
header, footer {
    background: #333;
    color: #fff;
    padding: 1rem;
}
nav ul {
    list-style: none;
    display: flex;
}
nav li {
    margin-right: 1rem;
}
nav a {
    color: #fff;
    text-decoration: none;
}
.container {
    display: flex;
    margin-top: 1rem;
}
aside {
    width: 25%;
    background: #f4f4f4;
    padding: 1rem;
}
main {
    width: 75%;
    padding: 1rem;
}

Despre fiecare stil:

  • display: flex; pe container permite poziţionarea pe orizontală.
  • Setările de lățime pentru aside și main definesc proporţiile.
