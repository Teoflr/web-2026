Săptămâna 7 – Introducere în PHP

PHP (Hypertext Preprocessor) este un limbaj de scripting folosit pe partea de server pentru a crea pagini web dinamice. În această săptămână vom învăța elementele fundamentale: variabile, tipuri de date, operatori, structuri de control și funcții simple. Toate exemplele rulează într-un mediu XAMPP (Apache + PHP).

1. Cum funcționează PHP?

Spre deosebire de HTML (care este client-side), codul PHP este executat pe server. Serverul procesează scriptul PHP și trimite în browser doar rezultatul (de obicei HTML). Astfel, utilizatorul nu vede niciodată codul PHP sursă.

  • Fișierele PHP au extensia .php (de exemplu index.php).
  • Codul PHP se scrie între etichetele <?php și ?>.
  • PHP poate fi amestecat cu HTML în același fișier.
Verificare instalare PHP: Creează un fișier info.php în htdocs cu conținutul <?php phpinfo(); ?>. Accesează http://localhost/info.php – vei vedea o pagină cu detalii despre configurația PHP.

2. Primul script PHP

În folderul htdocs (sau într-un subfolder al proiectului) creează fișierul hello.php:

<!DOCTYPE html>
<html lang="ro">
<head>
    <meta charset="UTF-8">
    <title>Primul script PHP</title>
</head>
<body>
    <?php
        echo "<h1>Bun venit în lumea PHP!</h1>";
        echo "<p>Astăzi este: " . date("d-m-Y") . "</p>";
    ?>
    <p>Acest paragraf este HTML pur.</p>
</body>
</html>

Accesează http://localhost/hello.php. PHP va executa comenzile echo și va afișa titlul și data curentă. Observă cum poți combina HTML static cu cod dinamic.

Explicații:

  • <?php ... ?> – delimitează codul PHP.
  • echo – construcție care afișează text în pagina finală (similar cu print).
  • date("d-m-Y") – funcție predefinită care returnează data curentă în formatul specificat (zi-lună-an).
  • Punctul . – operator de concatenare (lipire) pentru șiruri de caractere.

3. Variabile și tipuri de date

În PHP, variabilele încep cu semnul $ urmat de nume. Numele sunt case-sensitive ($nume ≠ $Nume). Nu este necesar să specifici tipul – PHP face automat conversia (limbaj tipizat dinamic).

<?php
    // String (șir de caractere)
    $nume = "Ana";
    $mesaj = 'Bună ziua!';
    
    // Integer (număr întreg)
    $varsta = 25;
    
    // Float (număr cu virgulă)
    $pret = 49.99;
    
    // Boolean (adevărat/fals)
    $student = true;
    
    // Afișare cu variabile
    echo "Nume: $nume <br>";
    echo 'Varsta: ' . $varsta . ' ani<br>';
    echo "Pret: {$pret} lei";
?>
Diferența dintre ghilimelele simple și duble: În șirurile cu ghilimele duble (" "), variabilele sunt interpretate. În șirurile cu ghilimele simple (' '), ele sunt afișate literal. De exemplu, "$nume" va afișa "Ana", pe când '$nume' va afișa "$nume".

Constante

Constantele sunt valori care nu se schimbă pe parcursul scriptului. Se definesc cu define() sau cu cuvântul cheie const.

<?php
    define("SITE_NAME", "Curs PHP");
    const AN_CURENT = 2026;
    echo SITE_NAME . " - " . AN_CURENT;
?>

4. Operatorii de bază

  • Aritmetici: + (adunare), - (scădere), * (înmulțire), / (împărțire), % (modul – restul împărțirii).
  • Atribuire: =, +=, -=, *=, /=.
  • Comparare: == (egal ca valoare), === (egal ca valoare și tip), !=, <>, !==, <, >, <=, >=.
  • Logici: && (și), || (sau), ! (non).
<?php
    $a = 10;
    $b = 3;
    $suma = $a + $b;      // 13
    $produs = $a * $b;    // 30
    $rest = $a % $b;      // 1
    
    $x = 5;
    $x += 2;              // $x devine 7
    
    $evaluare = ($a > $b) && ($suma == 13); // true
    echo "Suma este $suma, rest: $rest";
?>

5. Structuri de control

Structurile de control decid fluxul execuției programului în funcție de condiții sau repetă operații.

5.1. if, elseif, else

<?php
    $nota = 85;
    if ($nota >= 90) {
        echo "Excelent!";
    } elseif ($nota >= 70) {
        echo "Foarte bine!";
    } elseif ($nota >= 50) {
        echo "Suficient.";
    } else {
        echo "Mai multă muncă!";
    }
?>

5.2. switch

<?php
    $zi = "marți";
    switch ($zi) {
        case "luni":
            echo "Început de săptămână";
            break;
        case "marți":
            echo "A doua zi";
            break;
        default:
            echo "Altă zi";
    }
?>

5.3. Bucle

  • for – când știi de câte ori se repetă.
  • while – cât timp o condiție este adevărată.
  • foreach – pentru a itera prin array-uri.
<?php
    // for
    for ($i = 1; $i <= 5; $i++) {
        echo "Numărul $i <br>";
    }
    
    // while
    $contor = 0;
    while ($contor < 3) {
        echo "while: $contor <br>";
        $contor++;
    }
    
    // foreach cu array simplu
    $fructe = ["Măr", "Pară", "Prună"];
    foreach ($fructe as $fruct) {
        echo "Fruct: $fruct <br>";
    }
?>

6. Array-uri în PHP

Array-urile sunt structuri de date care pot stoca mai multe valori. PHP suportă array-uri indexate numeric, asociative (cu chei personalizate) și multidimensionale.

Array indexat numeric

<?php
    $culori = ["roșu", "verde", "albastru"];
    // sau $culori = array("roșu", "verde", "albastru");
    echo $culori[0]; // afișează "roșu"
    $culori[] = "galben"; // adaugă un element la final
?>

Array asociativ

Folosește chei text pentru a face codul mai expresiv.

<?php
    $student = [
        "nume" => "Alex Popescu",
        "varsta" => 22,
        "specializare" => "Informatică"
    ];
    echo $student["nume"]; // Alex Popescu
    
    // Parcurgere array asociativ
    foreach ($student as $cheie => $valoare) {
        echo "$cheie: $valoare <br>";
    }
?>

Funcții utile pentru array-uri

  • count($array) – numărul de elemente.
  • array_push($array, $val) – adaugă un element la sfârșit.
  • array_pop($array) – elimină ultimul element.
  • in_array($val, $array) – verifică dacă valoarea există.
  • sort($array) – sortează crescător.

7. Funcții definite de utilizator

Funcțiile ajută la organizarea codului și la reutilizarea logicii. Se definesc cu cuvântul cheie function.

<?php
    // Funcție simplă fără parametri
    function salut() {
        echo "Salutare din funcție!<br>";
    }
    salut();
    
    // Funcție cu parametri și return
    function aduna($x, $y) {
        return $x + $y;
    }
    $rezultat = aduna(12, 8); // 20
    echo "Suma este: $rezultat <br>";
    
    // Funcție cu valoare implicită
    function mesajBunVenit($nume = "vizitator") {
        return "Bine ai venit, $nume!";
    }
    echo mesajBunVenit("Elena"); // Bine ai venit, Elena!
    echo mesajBunVenit();        // Bine ai venit, vizitator!
?>
Return vs echo: O funcție care folosește return returnează o valoare, dar nu o afișează imediat. Valoarea poate fi stocată într-o variabilă sau afișată ulterior. echo în interiorul funcției trimite direct output-ul către browser.

8. Superglobalele și date din formular

PHP are variabile predefinite (superglobale) accesibile oriunde în script: $_GET, $_POST, $_SERVER, $_SESSION etc. Ele sunt folosite pentru a primi date de la utilizator sau informații despre server.

Definiții și mod de utilizare pentru fiecare superglobală

1. $_GET

Definiție: Array asociativ care conține parametrii transmisi prin URL (în string-ul de interogare). Se vede în URL ca ?param1=valoare1&param2=valoare2.

Caracteristici: Date vizibile în URL, limitate ca mărime (~2KB), mai puțin sigure. Ideal pentru filtrare, paginare sau parametri simpli.

Exemplu: procesare simplă cu GET

Creează două fișiere: formular.html și procesare.php.

<!-- formular.html -->
<form action="procesare.php" method="get">
    <label>Numele tău: </label>
    <input type="text" name="nume_utilizator">
    <button type="submit">Trimite</button>
</form>
<?php // procesare.php
    if (if($_GET['nume_utilizator'])) {  // Verificăm dacă există parametrul în URL - se poate si cu metoda isset($_GET['nume_utilizator'])
        $nume = htmlspecialchars($_GET['nume_utilizator']);
        echo "<h2>Bună, $nume!</h2>";
    } else {
        echo "<p>Niciun nume primit.</p>";
    }
?>
Securitate: Funcția htmlspecialchars() previne atacurile XSS (Cross‑Site Scripting) convertind caracterele speciale (<, >, ", ') în entități HTML. Folosește‑o întotdeauna când afișezi date primite de la utilizator.

2. $_POST

Definiție: Array asociativ care conține datele trimise prin metoda POST din formulare. Datele nu apar în URL, ci în body-ul cererii HTTP.

Caracteristici: Date ascunse, suportă fișiere și date mari, mai sigură decât GET. Ideală pentru formularele de login, înregistrare sau trimitere de date sensibile.

<?php
    // Din formular: <form method="POST">
    if ($_SERVER['REQUEST_METHOD'] == 'POST') {
        $parola = isset($_POST['parola']) ? $_POST['parola'] : '';
        $email = isset($_POST['email']) ? htmlspecialchars($_POST['email']) : '';
        if (!empty($email) && !empty($parola)) {
            echo "Login incercat cu: $email";
        }
    }
?>

3. $_SERVER

Definiție: Array asociativ care conține informații despre server și cererea HTTP actuală: adresa IP a clientului, metoda cererii (GET/POST), URL-ul paginii, header-e HTTP, etc.

Caracteristici: Doar citire (setată de server), conține metadate. Util pentru verificări de securitate și logging.

<?php
    echo "Metoda cererii: " . $_SERVER['REQUEST_METHOD'] . "<br>";
    echo "IP client: " . $_SERVER['REMOTE_ADDR'] . "<br>";
    echo "URL pagina: " . $_SERVER['PHP_SELF'] . "<br>";
    echo "Server: " . $_SERVER['SERVER_NAME'] . "<br>";
    echo "User-Agent: " . $_SERVER['HTTP_USER_AGENT'] . "<br>";
    
    // Verificare metoda POST
    if ($_SERVER['REQUEST_METHOD'] == 'POST') {
        echo "Formular trimis prin POST";
    }
?>

4. $_SESSION

Definiție: Array asociativ care stochează date persistent pentru un utilizator pe parcursul unor multiple cereri. Datele rămân în memorie pe server până când sesiunea se termină.

Caracteristici: Necesită session_start(), datele nu se trimit la client, sigure pentru informații sensibile (ID utilizator, preferințe). Fiecare vizitator are propria sesiune.

<?php
    session_start(); // Trebuie apelat la INCEPUTUL fișierului, înainte de orice output
    
    // Stocheaza date in sesiune dupa login
    $_SESSION['user_id'] = 123;
    $_SESSION['username'] = 'alexandra';
    $_SESSION['logged_in'] = true;
    
    // Pe alte pagini, retrieve datele
    if (isset($_SESSION['logged_in']) && $_SESSION['logged_in']) {
        echo "Bine ai venit, " . $_SESSION['username'];
    } else {
        echo "Va rugam sa va autentificati";
    }
    
    // Logout - sterge sesiunea
    // session_destroy();
?>

9. Exerciții practice

Pentru a fixa cunoștințele, încearcă să rezolvi următoarele exerciții:

  1. Calculator simplu: Creează un script PHP care primește două numere și un operator (+, -, *, /) prin metoda GET și afișează rezultatul. Tratează cazul împărțirii la zero.
  2. Listă de sarcini (To-Do): Folosește un array pentru a stoca 3-4 sarcini. Afișează-le într-o listă HTML folosind o buclă foreach. Adaugă un element nou în array cu array_push.
  3. Funcție de verificare paritate: Scrie o funcție estePar($numar) care returnează true dacă numărul este par și false în caz contrar. Apeleaz-o pentru mai multe valori și afișează mesaje corespunzătoare.
  4. Generare numere prime: Creează un script care afișează toate numerele prime între 1 și 100 folosind o funcție estePrim($n) și o buclă.
